# React Task Manager + User Search

## Live Demo

https://react-task-manager-beta-taupe.vercel.app

---

## Overview

A React-based task management application with integrated user search functionality. This project demonstrates core frontend concepts such as state management, API integration, and dynamic UI updates.

---

## Features

* Add, delete, and mark tasks as completed
* Persistent task storage using LocalStorage
* Fetch users from an external API
* Search and filter users dynamically
* Responsive and interactive UI

---

## Tech Stack

* React (useState, useEffect)
* JavaScript (ES6+)
* HTML & CSS

---

## Project Purpose

This project was built to strengthen frontend development skills and understand how to build interactive applications using React.

It focuses on:

* State management using hooks
* API data fetching and handling
* Component-based architecture
* User interaction and UI updates

---

## Application Workflow

1. Users can add and manage tasks
2. Tasks are stored in LocalStorage for persistence
3. User data is fetched from an API
4. Search functionality filters users in real time
5. UI updates dynamically based on user interaction

---

## How to Run Locally

1. Clone the repository:

   ```
   git clone https://github.com/ShubhamKabir/react-task-manager
   ```

2. Navigate to the project folder:

   ```
   cd react-task-manager
   ```

3. Install dependencies:

   ```
   npm install
   ```

4. Start the development server:

   ```
   npm start
   ```
